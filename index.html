So it's a Framework I suppose?

Tempaltie is what I like to call a wet-clay-web framework; it provides a strong foundation for a website's stylesheet that requires all parts to be looked at and altered to be correct for the use. Much like that scene in ghost; if you ignore the clay and get distracted thinking it will be fine, a Lionel Richie song will start to play and your site will look terrible*.

The way to use this is to add and remove files you need. So if you are site developer there is a helpful LESS file called code.less that will format code such as functions() in the correct font and even do blocks correctly!

function chinchilla() {
	$boom->getAviatorGoogles();
	return $boom.' Chinchilla';
}

But if you don't really need to display code on your site, then you wouldn't include it, so that removes all code styles.

* I never saw Ghost
